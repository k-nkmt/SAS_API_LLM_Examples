# SASにおける外部APIと自然言語の利用例

[SASユーザー総会2023](https://sas-user2023.ywstat.jp/) の発表内容に関するファイルを公開しています。

以下の内容のnotebookファイルを置いています。
- SASによる基本的なAPIの利用
- 関数呼び出し機能の利用
- Langchainを模したエージェント

エージェントについてはsasプログラムを読み込んで利用しています。  
他は単体で実行可能です。

## 準備
OpenAIのAPIキーを用意しておいてください。  
動作確認はSAS Ondemand(9.4_M7)で行いました。  
またSASのセッションエンコーディングは`utf-8`とします。  
一般的なAPIの知識や、OpenAIのAPIに関する詳細については必要に応じて別途確認しておいてください。
